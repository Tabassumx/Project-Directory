## 🛠 Featured Projects

### ☁️ AtmosWeather

A modern, luxury weather dashboard focused on layout stability and interactive visual elements.

- **Tech:** Vanilla JavaScript, Three.js, Tailwind CSS  
- **Key Features:** Real-time API integration, 3D environmental cues.

### ⚖️ BodyBuzz

A technical health metrics dashboard designed with a Neo-Brutalist aesthetic.

- **Tech:** JavaScript, Tailwind CSS  
- **Key Features:** Precise BMI calculations using the Mifflin-St Jeor equation and personalized 6-month health planning.

### 🌙 Salah Tracker

A spiritual productivity tool designed to help users monitor and improve their focus during prayer.

- **Tech:** JavaScript, Tailwind CSS  
- **Key Features:** Daily tracking, minimalist UI, and progress insights.

### 🧴 Liore Skin Care

A high-end skincare boutique interface emphasizing aesthetic elegance and seamless user experience.

- **Tech:** E-Commerce UI, Branding Design  
- **Key Features:** Smooth product exploration and premium visual storytelling.

### 👟 TXK Shoe

An experimental landing page for footwear, focusing on bold typography and motion design.

- **Tech:** Interactive Animations, Product Design  
- **Key Features:** Dynamic product cards and fluid layout transitions.
